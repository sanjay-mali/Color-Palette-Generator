Color Palette Generator

Generate a color palette using the color picker or generate a color scheme using the color wheel.
Click on the lock icon to lock a color in place. Click on the color box to unlock it.
Select a color harmony from the dropdown menu and click the "Generate Palette" button to generate a color palette.
Select a color scheme from the dropdown menu and click the "Generate Colors" button to generate a color scheme.
